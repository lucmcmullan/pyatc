1.0.0 - Initial release

    - Aircraft spawn and manual control
    - Console input for simple commands

    1.1.0 - Radio logging

        - Controller <-> pilot history
        - Sidebar for comms display

            - Improved message queue handling

    1.2.0 - Proper typing, command parsing, chaining & flight control
        - Implemented command parser with chaining
        - Flight control logic for heading, speed and altitude
        - Planes respond to parsed instruction sequences

        1.2.1 - Minor improvement
            - Heading updates not updating until next command fix

        - Added realistic ATC phraseology for acknowledgements

        - Added support for EX (expedite) responses

        1.3.1 - Parse refinements - EX fixes
            - Fixed EX not complying with chained messages
            - Enhanced phrasing for multi-command responses

        1.3.2 - Multi-aircraft command targeting

            - Cleaner controller-side feedback

    1.4.0 - Responsive layout, text wrapping
        - Dynamic layout scaling (WIDTH/HEIGHT-aware calculator)
        - Sidebar and radar sizes adapt to resolution

        1.4.1 - UI polishing
            - Fixed clipped text for small resolutions
            - Adjusted scroll behaviour and visual spacing

        1.4.2 - Standardised capitalisation

        - Introduced configurable climb-rate curve for acceleration

        1.5.1 - Altitude dynamic refinements
            - Fixed initial climb spike

            - EX doubles climb rate while retaining smooth motion

        1.5.2 - Stability improvements
            - Fixed zero-division in altitude interpolation
